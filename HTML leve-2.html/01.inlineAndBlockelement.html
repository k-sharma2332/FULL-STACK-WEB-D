Block Elements : These are the elements that occupy a whole line or block in a web page irrespective of the unused space. However, they can use more than one line if needed. These elements are always added in a new line.

  • Header Tag 1

    Header Tag 2

  • Paragraph Tag 1

    Paragraph Tag 2



Inline Elements : These are the elements that can be placed in a single line or block in a web page. These elements are added in continuation after one another and do not need a new line every time.
